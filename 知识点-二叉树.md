Python中的二叉树的表现形式
```Python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
```





深度优先搜索（DFS）
深度优先搜索（DFS）是一种遍历或搜索树或图的算法。它沿着树的深度进行搜索，直到找到目标节点或遍历完整个树。DFS可以使用递归或迭代的方式实现。
简单来说，DFS定一个方向（比如左边），就一直往前走，遇到岔路口：你依然选最左边的路走，直到走到头了，再回退到上一个岔路口，选第二条路走，依次类推，直到所有的路都走过了。

DFS的三个关键：
1.选择去哪儿，比如在二叉树结构里就是选择左子树还是右子树，在图论里就是选择哪个邻接节点
2.什么时候停止，比如在二叉树结构里就是当节点为None或者节点值满足条件时，在图论里就是当节点已经被访问过了
3.回退到上一个状态，继续选择其他的路走

可以使用递归的方式实现DFS：
```Python
def dfs(node: TreeNode):
    if not node:
        return
    # 处理当前节点
    print(node.val)
    # 递归遍历左子树
    dfs(node.left)
    # 递归遍历右子树
    dfs(node.right)
